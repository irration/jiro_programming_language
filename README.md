# プログラミング言語 二郎  
## 概要  
二郎はラーメンではなく二郎というプログラミング言語である。　　　   
(現段階ではプログラミング言語というよりただのコンパイラです)  

## 用例  
make コマンドにて 二郎語のコンパイラ(jiroparse)を生成する。  
`make`  

二郎語のコンパイラを実行して対象となるtest.jiroファイルをコンパイルする。  
`./jiroparse test.jiro`
